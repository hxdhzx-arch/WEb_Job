隐私政策

最后更新：2025 年 6 月

1. 我们收集什么

当您使用简历 AI 时，我们会处理以下信息：

• 您主动提交的简历文本：用于 AI 诊断、匹配和生成功能

• 上传的 PDF 文件：用于简历解析和结构化填充

• 账号信息：如您选择绑定，我们存储邮箱或手机号、算力余额、以及云端保存的简历数据

• 设备标识：我们使用 FingerprintJS 生成匿名设备指纹（visitorId），用于风控防刷和新手算力发放。该指纹不包含您的姓名、地理位置或浏览记录

2. 隐私防火墙技术（PII 脱敏）

在将您的简历内容发送给 AI 模型之前，系统会自动检测并替换以下个人身份信息（PII）：

• 手机号 → [手机已隐藏]

• 邮箱地址 → [邮箱已隐藏]

• 身份证号 → [证件号已隐藏]

• QQ 号、微信号 → [已隐藏]

脱敏后的文本被发送给 AI 模型处理，AI 模型无法获知您的真实联系方式。

3. 第三方 AI 模型处理

重要说明：您的简历文本（经脱敏处理后）会被发送至 Google Gemini API 进行 AI 分析和生成。这意味着经脱敏的文本会经过 Google 的服务器处理。我们依据 Google 的 API 服务条款使用该服务，Google 承诺 API 调用数据不会用于模型训练。

我们不会将您的数据出售给任何第三方，也不会将其用于广告投放或用户画像。

4. 数据存储

简历 AI 采用分层存储策略：

未绑定账号时简历数据仅保存在您的浏览器本地（localStorage）。AI 分析时，脱敏后的文本仅在服务器内存中短暂存在，分析完成后不做持久化存储。
绑定账号后如果您选择绑定邮箱或手机号，您的简历数据将保存到我们服务器的数据库中（SQLite），以便跨设备同步。您可以随时删除云端的任何一份简历。
上传的 PDFPDF 文件仅在解析期间存在于服务器内存中，解析完成后立即丢弃，不做持久化存储。

5. 设备指纹与追踪

我们使用 FingerprintJS（开源版）生成匿名设备标识符。该技术基于浏览器特征（屏幕分辨率、字体列表等）生成一个哈希值，不属于 Cookie，但在功能上类似于设备标识。我们将其用于：

• 防止同一设备重复领取新手算力

• 风控识别异常注册行为

我们不使用 Google Analytics、百度统计等第三方数据分析追踪服务。

6. 您的权利

您可以随时：

• 删除本地数据：在简历编辑器中点击「清除所有数据」按钮

• 删除云端简历：在简历列表中逐条删除，或联系我们要求删除全部云端数据

• 注销账号：联系我们删除您的用户记录、算力余额和所有关联数据

• 拒绝设备指纹：使用浏览器隐私模式可使指纹每次会话不同，但这可能导致算力重复消耗

7. 数据安全

我们采取以下措施保护您的数据：

• PII 脱敏引擎在数据离开服务器前自动运行

• API 通信使用 HTTPS 加密传输

• 数据库文件存储在服务器本地，不使用公有云对象存储

• 验证码有效期 5 分钟，错误 5 次自动作废

8. 联系我们

如有隐私相关问题，请通过页面底部联系方式与我们联系。我们承诺在 7 个工作日内响应数据删除请求。
